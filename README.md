# ImageProcessingApp 

## 📕 Opis
ImageProcessingApp to aplikacja służąca do podstawowego przetwarzania obrazów. Program umożliwia ich wczytywanie, zapisywanie, analizowanie oraz modyfikowanie na różne sposoby.

## 💻 Użyte technologie
- C# WinForms
- biblioteka MaterialSkin
